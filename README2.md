AI-powered text leveling tool for Chinese heritage speakers and near-native learners.

## 功能 / Features

- **双语支持 / Bilingual**: 支持中文和英文文本分级 / Supports both Chinese and English text leveling
- **即时分级 / Instant Differentiation**: 粘贴文本，生成三个难度版本 / Paste text, get scaffolded/standard/advanced versions
- **标准化对接 / Standardized**: 基于 Lexile/CEFR/HSK 标准开发中 / Developing based on Lexile/CEFR/HSK standards

---

## 使用方法 / How to Use

1. 访问链接 / Visit the link
2. 在输入框粘贴中文或英文文本 / Paste Chinese or English text in the input box
3. 点击"生成" / Click "Generate"
4. 获取三个分级版本 / Get three differentiated versions

---

## 技术栈 / Tech Stack

- Frontend: HTML/CSS/JS
- Backend: Node.js + Express
- AI Tool: Trae/ OpenAI/ Claude / Minimax API
- Deployment: Railway
